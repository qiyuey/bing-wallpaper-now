# Changelog

## 1.5.2

### Changed

- 设置页现在会显示已保存壁纸数量和日期范围，打开目录、更改目录、导入、导出操作也更集中清晰
- 界面卡片、弹窗和按钮的视觉层级与动效更克制，并会跟随系统减少透明度和减少动态效果偏好
- 更新应用图标，桌面、安装包和浏览器预览中的图标保持一致

### Fixed

- 修复启动时可能短暂显示错误主题的问题，应用会更早应用上次选择的浅色、深色或系统主题偏好

## 1.5.1

### Changed

- 设置页控件尺寸与按钮样式更统一，导入/导出等操作区更紧凑一致

### Fixed

- 修复 Windows 上玻璃材质可能失效的问题，卡片、按钮和弹窗的模糊效果恢复正常

## 1.5.0

### Added

- 全新 Liquid Glass 设计语言，界面更通透，壁纸色彩贯穿整个应用
- 新增 SegmentedControl 分段选择器组件，带平滑滑动动效
- 主页背景显示壁纸模糊氛围色
- 应用截图加入 README

### Changed

- 卡片改为半透明玻璃材质，背景色随壁纸变化
- 导航栏改为浮动 Liquid Glass 样式
- 设置弹窗使用类 Windows Mica 效果（高模糊 + 色调渲染）
- 主题色从 Indigo 改为 Apple Blue (#007AFF)
- 复选框改为 iOS 风格 Toggle 开关
- 设置标签去除冒号
- 关于页面去除底部关闭按钮（保留右上角 X）
- 窗口最小宽度提升至 520px

### Fixed

- 点击"设置壁纸"按钮不再误触发图片链接跳转
- 深色模式下拉选项背景和文字对比度改善
- 窄窗口时自动隐藏更新时间信息
- 无障碍：分段选择器 radio 添加 name 属性

## 1.4.9

### Fixed

- 修复从托盘手动检查更新时，无更新也可能报错或打开主窗口的问题；现在无更新会发送通知，有更新才显示更新窗口

## 1.4.8

### Changed

- macOS 版本改为仅提供 Apple Silicon 构建

## 1.4.7

### Fixed

- 修复部分平台或启动方式下自动语言检测不准确的问题

## 1.4.6

### Fixed

- 修复通过 Homebrew Cask 安装时出现的弃用语法警告

## 1.4.5

### Added

- 新增 [AI_INSTALL.md](AI_INSTALL.md)，为 AI 助手 / 命令行代理提供覆盖 macOS / Windows（含 ARM64）的可复制即运行安装命令

### Changed

- 重新规划卡片网格间距：列间距、行间距、容器边距由统一变量驱动，所有屏幕尺寸下视觉一致
- MacBook Pro 16 寸全屏窗口默认保持 3 列，仅外接 4K 等更宽显示器才切换到 4 列布局

### Fixed

- 修复滚动列表或调整窗口宽度时，卡片之间偶尔出现的视觉残影
- 修复窄屏与双列布局下，卡片之间行间距意外消失的问题
- 修复某些窗口宽度下卡片排列出现"3+1"等不规则换行的问题
- 修复 Winget 渠道可能附带非 MSI 安装包的问题，确保 Windows 用户通过 Winget 安装到正确的安装器

## 1.4.4
