# Changelog

## 1.4.9

### Fixed

- 修复从托盘手动检查更新时，无更新也可能报错或打开主窗口的问题；现在无更新会发送通知，有更新才显示更新窗口

## 1.4.8

### Changed

- macOS 版本改为仅提供 Apple Silicon 构建

## 1.4.7

### Fixed

- 修复部分平台或启动方式下自动语言检测不准确的问题

## 1.4.6

### Fixed

- 修复通过 Homebrew Cask 安装时出现的弃用语法警告

## 1.4.5

### Added

- 新增 [AI_INSTALL.md](AI_INSTALL.md)，为 AI 助手 / 命令行代理提供覆盖 macOS / Windows / Linux（含 ARM64）的可复制即运行安装命令

### Changed

- 重新规划卡片网格间距：列间距、行间距、容器边距由统一变量驱动，所有屏幕尺寸下视觉一致
- MacBook Pro 16 寸全屏窗口默认保持 3 列，仅外接 4K 等更宽显示器才切换到 4 列布局

### Fixed

- 修复滚动列表或调整窗口宽度时，卡片之间偶尔出现的视觉残影
- 修复窄屏与双列布局下，卡片之间行间距意外消失的问题
- 修复某些窗口宽度下卡片排列出现"3+1"等不规则换行的问题
- 修复 Winget 渠道可能附带非 MSI 安装包的问题，确保 Windows 用户通过 Winget 安装到正确的安装器

## 1.4.4

### Changed

- 长时间网络中断后，自动更新会以 15 / 30 / 60 分钟阶梯式重试当日壁纸，无需再等到下一个整点

### Fixed

- 修复多屏环境下使用横屏壁纸代替缺失的竖屏壁纸时，仍误报"部分显示器壁纸设置可能失败"的问题

## 1.4.3

### Fixed

- 修复发布后短暂时间内应用内更新可能检测到新版但无法下载的时序问题

## 1.4.2

### Fixed

- 修复 CI 发布构建中 Rust 缓存始终未命中的问题，显著缩短发布构建时间

## 1.4.1

### Fixed

- 修复正式版中点击"检查更新"后窗口打开但无任何通知反馈的问题

## 1.4.0

### Added

- 支持通过 Homebrew Cask 安装 (macOS)：`brew tap qiyuey/tap && brew install --cask bing-wallpaper-now`

## 1.3.2

- 测试自动更新流程

## 1.3.1

### Fixed

- 修复应用内自动更新无法正常工作的问题

## 1.3.0

### Added

- 应用内更新：支持直接下载安装新版本，带进度显示和自动重启

### Changed

- 移除首页动态标语
